cara nya


1. harus ada phyton 
2. install virtual envnya, abis tu buka cmd di folder ini, ketikin python -m venv venv
3. absi tu buka virtual envnya, caranya ketikin di cmdnya venv\scripts\activate
4. install module nya, caranya ketikin di cmdnya pip install flask pandas numpy scikit-learn
5. setelah selesai, start servernya, dicmdnya ketikin python app.py
6. setelah ada pemberitahuan server telah dimulai, buka http://127.0.0.1:5000/
